 ## Project E-commerce

## Installation

```bash
$ npm install
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Agusdor96/agus-ecommerce.git
```
Install dependencies

```bash
  npm install

  DEVELOPMENT
  npm run start

  WATCH MODE
  npm run start:dev

  PRODUCTION MODE
  npm run start:prod
```
